### 2장. **디렉터리**
#### 2-1 파일의 종류
1. 일반 파일 - 텍스트 파일, 실행 파일, 라이브러리 파일
2. 특수 파일 - 장치 파일(디바이스 파일 - 블록 디바이스, 캐릭터 디바이스)
3. 디렉터리 - 폴더

* 파일 종류 구분  
ls -l 명령 출력 결과의 맨 앞 문자  

|문자|파일 종류|
|:--:|--|
|-|일반 파일|
|d|디렉터리|
|b|블록 장치 특수 파일|
|c|문자 장치 특수 파일|
|l|심볼릭 링크|
